# DermCare - Skincare Management Application

**DermCare** is a cross-platform skincare management application built using **React** for web and mobile, powered by the **MERN stack**. It helps users create personalized skincare routines, track their progress, and access educational content about skincare.  

## Features

- **Personalized Skincare Routines**: Users can create routines tailored to their skin type and goals.  
- **Progress Tracking**: Track improvements and monitor skincare milestones over time.  
- **Educational Content**: Access tips, articles, and guidance on skincare practices.  
- **Cross-Platform Support**: Works seamlessly on both web and mobile devices.  
- **Secure and User-Friendly**: Focus on a smooth, secure, and intuitive user experience.  

## Tech Stack

- **Frontend**: React, React Native, Redux  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Authentication**: JWT-based secure login  
- **Styling**: CSS Modules / Tailwind CSS
